CMT - A Hardness Testing System

Technosys offers the most practical Automatic Micro Vickers Hardness Tester Clemex CMT. The Robost tester can be used in continuous operation 24/7 and 365 days, proven track record of the system. This system is perfectly suitable for taking hardness on thin samples, coating, plating, Case hardened, induction hardened, Nitriding parts etc.. Apart from the regular components. Clemex CMT can produce accurate results on ECD / EHD with graph in automatic manner with high accuracy and repeatability. Operation is as simple as 1, 2, 3… Load the sample, select the test pattern, Start the Run the machine gives the results with report.
